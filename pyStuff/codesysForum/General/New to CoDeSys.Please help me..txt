It's easy to just toggle blinking of e.g. a led. Try this code in your project.

Code:PROGRAM PLC_PRGVAR   Button1 AT %IX0.0   : BOOL;          (* A button connected to first digital (binary) input *)   Led1 AT %QX0.0      : BOOL;          (* A led connected to first digital (binary) output *)   Blink1              : BLINK;         (* Make an instance of Blink FB. Found in Utils.lib *)   Trig1               : R_TRIG;        (* Rising edge detection. Found in Standard.lib *)   xBlink              : BOOL := FALSE; (* TRUE=Blinking enabled, FALSE=Blinking disabled *)END_VAR---Trig1(CLK:=Button1);IF Trig1.Q THEN                         (* If detected a rising edge *)   xBlink := NOT xBlink;                (* Toggle blink state *)        IF NOT xBlink THEN                        Blink1(ENABLE:=FALSE);        (* Reset blink routine *)          Led1 := FALSE;                (* and output. Doing a OUT=>Led1 does not guarantee to switch led off *)               END_IFEND_IFIF xBlink THEN   Blink1(ENABLE:=TRUE, TIMELOW:=t#500ms, TIMEHIGH:=t#500ms, OUT=>Led1);   END_IF
