Guten Tag Olaf,Codesys Webvisu funktioniert problemlos in einem lokalen Netzwerk, sodass Sie sich darüber keine Sorgen machen müssen. Hier ist, was Sie sehen müssen.1. Sie müssen einen Port auf Ihrem Router öffnen, der Sie mit dem Internet verbindet. Jeder Router ist anders, also finden Sie Anweisungen im Netz. Dann müssen Sie diesen Port an Ihre Himbeer-Pi weiterleiten.2. Überprüfen Sie, ob Ihre Himbeere mit dem Internet verbunden ist, indem Sie curl ipecho.net/plain;echoDieser Befehl wird Ihr externes IP wiedergeben. Notiere diese IP-Adresse.3. Verwenden Sie auf Ihrem Smartphone den Webbrowser und geben Sie die angegebene IP-Adresse ein, z. B. 192.168.2.100:8080/webvisu.htm. Wenn Sie Ihre Seite sehen, haben Sie das richtig gemacht.4. Sie können dann ein Konto bei einem ddns-Anbieter registrieren, no-ip verfügt über eine Software, die Sie auf der Himbeere ausführen können, um diesen Dienst bereitzustellen. Dies hat den Vorteil, dass der Dienst beim Neustart oder beim Einschalten des Dienstes dynamisch verbunden wird zum Service und Sie verlieren Ihre Verbindung nicht, weil Sie eine ddns Adresse haben, die zu Ihrer Himbeere bestimmt ist.Es könnte etwas mehr Forschung erfordern, um alles in Ordnung zu bringen.Bitte entschuldigen Sie die Deutsch, aber ich benutze eine net-basierte Übersetzung aus dem Englischen.Ich hoffe, dass ich Sie in die richtige Richtung hingewiesen habe.Freundliche Grüße,Johan
