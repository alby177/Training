To move from one display page to another you must use a button in each page with the following settings :In the page called "PLC_VISU" :- Category Text : for example the name of the page you want to go, or one another text- Category Input : Check the box "_ to vis. : and with the F2 Function Key, choose the name of the page you want to switch on "second_page"In the page called "second_page" :- Category Text : for example the name of the page you want to go, or one another text- Category Input : Check the box "_ to vis. : and with the F2 Function Key, choose the name of the page you want to switch on "PLC_VISU"Webvisu.htm is a file in your controler. But you do not need to change it to do what is explained in the previous lines.Hope this helps you
